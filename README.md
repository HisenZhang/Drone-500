## Utilities

### resize

```
usage: resize.py [-h] [--output OUTPUT] input h w

positional arguments:
  input            Input file path.
  h                Height.
  w                Width.

optional arguments:
  -h, --help       show this help message and exit
  --output OUTPUT  Output file path.
```
